Довідка системи TrainSorty

Інструкція з експлуатації та технічні стандарти сортування

📖 Порядок роботи з програмою

  • 1. Налаштування: Натисніть ⚙️, щоб додати міста та типи вагонів.
  • 2. Введення даних: Завантажте JSON-файл або введіть 8 цифр вручну.
  • 3. Валідація: Система автоматично перевірить наявність колії для вашого номера.
  • 4. Сортування: Натисніть 🚀 для автоматичного розподілу.
  • 5. Звітність: Після завершення сформуйте файл звіту для архіву.

🔢 Розпізнавання номерів вагонів (УЗ)

Тип вагона визначається за першою цифрою восьмизначного номера:

Перша цифра Рід рухомого складу Опис для системи
0, 1 Пасажирські Вагони для перевезення пасажирів та багажу.
2, 3, 5, 6 Вантажні Криті вагони, платформи та власні вагони підприємств.
7 Цистерни Спеціалізовані вагони для рідких вантажів.
4 Контейнерні Платформи для перевезення великотоннажних контейнерів.
8, 9 Технічні Рефрижератори, колійні машини та снігоочисники.
